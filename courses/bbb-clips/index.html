Each module slot has 1-4 candidate clips; you pick the winner (or ask for a different cut).

The embedded player starts at the proposed in-time and stops at the out-time. Click ↗ open on YouTube to scrub around the candidate in context (the player may need a manual seek if YouTube ignores the end-time parameter — that's a known YT IFrame quirk, not a bug here).

Reply with thumbs / no / "trim to start at X" / "go longer" for each clip you want to act on. Anything you don't comment on stays in the candidate pool for the next review pass.
