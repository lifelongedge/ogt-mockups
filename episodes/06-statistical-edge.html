02 / Read The principles

  1. Expectancy is the only number that matters.(Win rate × Avg win) − (Loss rate × Avg loss) − Costs. Positive expectancy is the edge; nothing else is.
  2. Win rate alone is not an edge.A 70% win rate system can be unprofitable if average loss > 2× average win. A 30% win rate system can be highly profitable with 4R winners.
  3. Sample size ≥ 100 trades before you trust the math.Below that, you're looking at noise. The variance of small samples obscures the true expectancy.
  4. Recompute monthly.Rolling expectancy across 30-trade windows catches degradation early. Edges decay; the monitoring catches it.
